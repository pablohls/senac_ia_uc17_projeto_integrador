## 📘 Guia de Desenvolvimento (para o integrante)

> Dificuldade 🟡 — peça revisão ao final. Conceitos repetidos nos glossários anteriores.

### 🎯 O que você vai construir
Para cada assunto (tópico), um "gráfico de quantas notícias por dia" — uma linha do tempo. Essa série é a matéria-prima para medir se um assunto está crescendo (a próxima story calcula o score em cima disso).

### 🧠 Conceitos que você precisa entender
- **Série temporal:** uma sequência de valores ao longo do tempo (aqui: nº de notícias por dia, por tópico).
- **Zero-fill:** se num dia o tópico não teve nenhuma notícia, o valor daquele dia é **0** (não "faltando"). Isso é essencial — sem o zero, as contas de crescimento da próxima story dão erro.
- **Resample / agregação semanal:** "juntar" os dias em semanas. Útil para tópicos com poucas menções (esparsos), onde o dia-a-dia é muito ruidoso.

### 🪜 Passo a passo
1. **Carregue** (Task 1): leia `doc_topics.parquet`.
2. **Conte por dia** (Task 2): `groupby(["topic_id", "data"]).size()`. Depois **reindexe** cada tópico para o intervalo completo de datas do corpus, preenchendo dias faltantes com 0. *Por quê:* a série precisa ser contínua (todos os dias), senão "crescimento" e "z-score" calculam errado.
3. **Some por semana** (Task 3): use `resample("W")` para gerar `count_weekly`. *Por quê:* dá uma visão mais estável para tópicos com poucos dados.
4. **Salve** (Task 4): grave `series.parquet`.

### ✅ Como saber que funcionou
- `series.parquet` tem `{topic_id, data, count}` cobrindo **todos** os dias do período (com zeros). *(AC1, AC3)*
- Existe `count_weekly` para a visão semanal. *(AC2)*
- Para um tópico, somar `count` = nº de notícias daquele tópico no corpus.

### ⚠️ Erros comuns & soluções
- **Dias faltando (sem zero):** esqueceu o reindex/zero-fill — a próxima story vai quebrar. Confirme que todo dia do período existe.
- **Datas como texto:** garanta que `data` é tipo data (`pd.to_datetime`) antes de agrupar/resample.

### 🆘 Quando me chamar
- Para decidir o intervalo exato do período (início/fim) usado no reindex.

### 📚 Glossário (novos termos)
| Termo | Significado simples |
|---|---|
| Série temporal | Valores ao longo do tempo |
| Zero-fill | Preencher com 0 os dias sem dado |
| Resample | Reagrupar por outra granularidade (dia→semana) |
